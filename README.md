# Relativistic Boats
## Mo Spiegel

&nbsp;

## Project Proposal
This project will be an interactive simulator that allows a user to input a velocity of a moving object, and then see the time dilation (from the reference frame) occurring on that object via a clock that ticks in accordance with the laws of Special Relativity. The program will feature a sleek UI, customizability options, and perhaps Lorentz Contraction of objects as well to further provide an intuitive grasp of relativistic phenomena.
